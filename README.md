# ppa_app_systemic_reduction
Replication package for the paper on Autoreferential Protection. Includes EV-02C and EV-04 simulation code, configuration files, analysis scripts, and processed datasets for mortality, stability, perturbation, and Granger causality. Raw ticks archived offline; full pipeline reproducible.
